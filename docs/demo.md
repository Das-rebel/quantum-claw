## 💡 Use Case Demos

### 🤖 Chatbot Demo
**Experience a multi-language chatbot:**

```bash
curl -X POST https://demo.quantum-claw.dev/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello! How can I help you today?",
    "conversationId": "demo-session-123"
  }'
```

### ✍️ Content Generation Demo
**Generate blog posts, articles, and more:**

```bash
curl -X POST https://demo.quantum-claw.dev/api/query \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Write a 500-word blog post about AI in healthcare",
    "options": {
      "provider": "openai",
      "temperature": 0.8
    }
  }'
```
